<Note>
  Esta API no tiene relación directa con el SII.
</Note>

## API Authentication

Para utilizar esta API es necesario disponer de un token y enviarlo en las cabeceras:

```json
"headers": [
  {
    "Authorization": "Bearer <token>"
  }
]
```

## SII Authentication

La mayoría de los servicios en esta API necesitan iniciar sesión en el SII. Por esto es necesario enviar los datos de acceso a la cuenta en el `body` de la solicitud:

```json
"body": [
  {
    "auth": {
      "pass": {
        "rut": "<string>",
        "clave": "<string>"
      }
    }
  }
]
```
